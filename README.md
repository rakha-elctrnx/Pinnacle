# Pinnacle

**Data Explorer for developers** — fast, local, and private.

A desktop application built with Tauri + React for exploring and managing your databases.

## Features

- **Multi-database support** — PostgreSQL, MySQL, MongoDB, Redis, RabbitMQ, Elasticsearch
- **SQL Query Editor** — Write and execute queries with syntax highlighting
- **Table Browser** — Browse table structures and data
- **Connection Management** — Save and manage multiple database connections
- **Local & Private** — All data stays on your machine

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Tauri (Rust)
- **Build:** Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri:dev

# Build for production
npm run tauri:build
